# AI Agents Portfolio

A collection of agentic AI projects built with LangChain and LangGraph — exploring tool use, multi-step reasoning, memory, and error handling in autonomous AI systems.

## Why This Repo Exists

I started this repo to build hands-on experience with agentic AI — systems that can reason through multi-step tasks, decide which tools to use, and recover gracefully when something goes wrong. This goes beyond single-shot prompting into building AI that can actually take autonomous action.

## What's Inside

Projects are organized by week as I build them out:

### Week 1 — Foundations
- Basic LangChain agent with tool selection (calculator + search)
- Basic LangGraph agent (same task, different framework)

### Week 2 — Real Tool Integration
- Agent connected to a live API with multi-tool decision logic
- Error handling and fallback logic for failed API calls

### Week 3 — Multi-Step Reasoning
- Agent that chains search → summarize → format into a final output
- Agent with persistent memory across multi-turn tasks

### Week 4 — Applied Project
- A complete agent project tied to a real use case (TBD — likely a research assistant or document Q&A agent)

## Tech Stack

- **Frameworks:** LangChain, LangGraph
- **LLM:** Claude API (Anthropic)
- **Tools:** Tavily Search API, Open Food Facts API
- **Vector Store:** ChromaDB
- **Language:** Python
